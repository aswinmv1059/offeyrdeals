- **Vanilla JavaScript** - No frameworks, pure ES6+ JavaScript
- **LocalStorage API** - Data persistence

## How to Use

1. **Add a task**: Type in the input field and click "Add Task" or press Enter
2. **Complete a task**: Click the checkbox next to the task
3. **Delete a task**: Click the trash icon on the right
4. **Filter tasks**: Use the buttons (All, Active, Completed) to filter your view
5. **Clear completed**: Click "Clear completed" to remove all completed tasks
